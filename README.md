# Loungebar 37 🍔🥩

A stunning, interactive restaurant website for Loungebar 37, featuring a 3D spinning burger carousel and an interactive steak showcase. Built with HTML, CSS, and JavaScript.

![Loungebar 37 Preview](https://via.placeholder.com/1200x630/F39C12/0A0A0A?text=Loungebar+37+Preview)

## ✨ Features

### 🍔 Interactive Burger Spinner
- 3D rotating carousel showcasing your signature burgers
- Manual controls with left/right arrows
- Click any burger to reveal:
  - High-resolution image
  - Complete ingredient list
  - Pricing information
- Smooth animations and transitions

### 🥩 Steak Showcase
- Interactive grid layout for premium cuts
- Hover effects with "Sizzle" animation
- Each steak card displays:
  - Cut name and description
  - Weight and price
  - Visual feedback on interaction

### 🎨 Design
- **Color Scheme:** Black (#0A0A0A) and Mandarin Orange (#F39C12)
- **Typography:** Oswald (headings) and Roboto (body text)
- Fully responsive design for mobile, tablet, and desktop
- Modern, sleek restaurant aesthetic
